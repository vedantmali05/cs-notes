Definition - Microprocessor

The Microprocessor also known as processor aka CPU (Central Processing Unit) is an IC (integrated circuit) acting as the central unit of a computer system that performs arithmetic and logic operations.

Arithmatic and Logical operations include addtion, subtraction, transfer of numbers from one area to another, and comparing two numbers. These operations are done in ALU (Arithmatic Logic Unit)

8086 Microprocessor is version of Microprocessor which was first designed by Intel in 1978. 8086 microprocessor has 20 Bit address bus. A Memory reads binary information which has 2 bits. Hence, 8086 has 220 ( 2 bits to the power 20-bit address bus) i.e. 1 MB addresses in the memory. 8086 microprocessor can perform arithmatic and logical operation on 16 bit data in one cycle.

Salient features of 8086 Microprocessor

  • Clock speed range of 5-10 MHz

Architecture of 8086

Functional Block Diagram

8086 Microprocessor is divided into 2 blocks - Bus Interface Unit (BIU) and Execution Unit (EU). Every component of 8086 microprocessor is present in these 2 blocks.

1) Bus Interface Unit (BIU)

BIU has a Instruction Stream Byte Queue which contains the set of instructions to be executed. This queue pre-fetches upto 6 bits and enqueue them. When one instruction is executed, the control unit needs to wait for fetching the next instruction. But when the instructions are pre-fetched, the speed of execution increases. BIU is the only part of microprocessor which is connected to memory and is connected by segment registers . These segment registers work with 16 Bits of binary data.
